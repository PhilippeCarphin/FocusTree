Help

ID is the number displayed in square brackets on the left of the tasks and TEXT is the text that will become the title of a new task. NOTES will become the closing notes of the task that will be visible as a tooltip when hovering over the task in the tree view.
treeGet a printable tree
currentGet a printable list of ancestors
next-task TEXTAdd a new sibling to the current task
new-task TEXTCreate a new root task
subtask TEXTCreate and enter a new subtask of the current task
done NOTESMark the current task as done (with optional closing notes)
helpShow the list of commands and their descriptions
subtask-by-id ID TEXTCreate a subtask of task with the specified id
delete-task ID Delete a task by id
info [ID] Print info on a task (if ID not specified, then current task)
switch-task IDSwitch to the task with the specified id
resetClear the whole tree
reassign-idsReassign all the ids in the case that they got messed up.
not-doneMark the current task as not doen
